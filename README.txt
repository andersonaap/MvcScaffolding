
1) Criar a solução

2) Adicionar o projeto MVC3 Razor

3) Rodar no Package Manager Console (NuGet)
PM> Install-Package MvcScaffolding

4) Criar um model
EX: Categoria e Receita

5) Compilar

6) Rodar no Package Manager Console (NuGet)
PM> Scaffold Controller <nome da model>

7) 
Scaffold Controller Categoria
Scaffold Controller Receita

8) Para sobrescrever
Scaffold Controller Categoria -Force
Scaffold Controller Receita -Force
